````md
# Kevin’s Framework (KRF)

## Prerequisites

- Rokit installed

## Setup

Run these commands from the repository root:

```powershell
rokit install
wally install
```
````

## Rojo

Serve the project:

```powershell
rojo serve default.project.json
```

Build a place file:

```powershell
rojo build default.project.json -o KRF.rbxl
```

## Tests

Run the test script:

```powershell
.\scripts\test.ps1
```
