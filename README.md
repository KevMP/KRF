# Kevin's Roblox Framework (KRF)

## Prerequisites
- Rokit installed
- Node and npm installed

## Setup
Run these commands from the repository root:

```powershell
rokit install
wally install
npm install
```

## Rojo
Serve the project:

```powershell
rojo serve default.project.json
```

Build a place file:

```powershell
rojo build default.project.json -o KRF-Tests.rbxl
```
